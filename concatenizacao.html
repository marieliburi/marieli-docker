  • O que é Docker
  • Conteinerização

A conteinerização é o processo de empacotar uma aplicação e todas as suas dependências em um container. Isso garante que o software funcione de forma consistente em qualquer ambiente: seja no computador do desenvolvedor, em servidores ou na nuvem.

Exemplo do mundo real

Imagine uma aplicação que usa um banco de dados MySQL. Sem Docker, você teria que instalar o MySQL manualmente. Com Docker, basta rodar: 

docker run -d -p 3306:3306 --name meu-banco -e MYSQL_ROOT_PASSWORD=123 mysql:latest

Em segundos, você terá um banco de dados funcional em um container, sem complicações.

Principais vantagens

  • Execução consistente em diferentes ambientes.
  • Redução de erros causados por configurações diferentes.
  • Mais rapidez para testes e deploys.
