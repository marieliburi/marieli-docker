O Docker é uma plataforma de código aberto que facilita a criação, execução e gerenciamento de aplicações dentro de containers. Ele funciona como uma "caixa" que empacota tudo o que sua aplicação precisa: código, bibliotecas, dependências e até configurações.

Por que usar Docker?

  • Portabilidade: o mesmo container pode rodar em qualquer máquina, sem precisar reconfigurar.
  • Eficiência: containers usam menos recursos que máquinas virtuais.
  • Escalabilidade: é fácil criar várias instâncias da mesma aplicação.

Exemplo prático

Imagine que você desenvolveu um site em Python Flask. Para rodar com Docker:

# Executar um container com Flask
docker run -p 5000:5000 flask-app

Pronto! O site vai rodar no navegador acessando http://localhost:5000.
